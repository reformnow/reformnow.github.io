# Agent Notes & Project Rules

## Bilingual Post Guidelines
- **Structure**: English paragraph followed by Chinese paragraph.
- **Div Wrappers**: 
    - English: `<div class="lang-en" markdown="1">...</div>`
    - Chinese: `<div class="lang-zh" markdown="1">...</div>`
- **Markdown Support**: The `markdown="1"` attribute is MANDATORY for the content to render correctly.
- **Bible Blocks**: Use ` ```bible ` for all scripture citations. Provide both languages.
- **Goal**: Maintain 15-minute read depth for theological articles with implicit Reformed insights.

## Theological Framework
- **Orientation**: Reformed (Implicit).
- **Key Concepts**: Sovereignty of God, Total Depravity, Unconditional Election, Limited Atonement (Implicit), Irresistible Grace, Perseverance of the Saints.
- **Tone**: Reverent, academic but accessible, meditative.

## Site Features
- **Timeline**: Data stored in `_data/timelines/`, images in `assets/img/timeline/`.
- **Bilingual Switcher**: Enabled by the presence of `lang-en` and `lang-zh` classes in the post content.
