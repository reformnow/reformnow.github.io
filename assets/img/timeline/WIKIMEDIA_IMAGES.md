## üîç **HOW TO SEARCH ON WIKIMEDIA COMMONS**

### Method 1: Direct URL
```
https://commons.wikimedia.org/wiki/File:[IMAGE_NAME]
```

### Method 2: Search Box
1. Go to: https://commons.wikimedia.org
2. Type in search box: `[Event name] painting`
3. Example: "Council of Nicaea painting"
4. Filter by: "Featured pictures" or "Quality images"

### Method 3: Browse Categories
- https://commons.wikimedia.org/wiki/Category:Ecumenical_councils
- https://commons.wikimedia.org/wiki/Category:Reformation
- https://commons.wikimedia.org/wiki/Category:Protestant_Reformation

## üì• **DOWNLOAD & PROCESS INSTRUCTIONS**

### Step 1: Download
1. Click the Wikimedia Commons link
2. Right-click on the image
3. Select "Save image as"
4. Save to your Downloads folder temporarily

### Step 2: Rename
Rename the file to match the required filename:
```
pentecost.jpg ‚úì
Pentecost_by_Titian.jpg ‚úó
```

### Step 3: Resize (Required)
Timeline images should be **800x400 pixels** (2:1 ratio)

**Option A: Online (Easiest)**
1. Go to: https://www.iloveimg.com/resize-image
2. Upload image
3. Set to: 800 x 400 pixels
4. Download

**Option B: ImageMagick**
```bash
convert input.jpg -resize 800x400^ -gravity center -extent 800x400 output.jpg
```

**Option C: Photoshop/GIMP**
- Crop to 2:1 ratio
- Resize to 800px width

### Step 4: Optimize
Reduce file size for web:
- **Goal:** Under 200KB per image
- **Tool:** https://tinypng.com
- Upload ‚Üí Compress ‚Üí Download

### Step 5: Move to Timeline Folder
```bash
mv ~/Downloads/pentecost.jpg /Users/kanglinyao/Projects/Jekyll-Sites/reformnow.github.io/assets/img/timeline/
```

### Step 6: Rebuild Site
```bash
cd /Users/kanglinyao/Projects/Jekyll-Sites/reformnow.github.io
bundle exec jekyll serve
```
